# Crypto-Currency-Tracking-Angular
The sample demonstrates how to use Wijmo interops with Angular. It implements a crypto currency tracking dashboard, with dynamic data updates, where data is represented using grids and charts.


## How to run the demo
- Clone the repository 
- Run *npm install* to install required libraries
- Run *npm run start*
